## Setup

インストール

```bash
# yarn
yarn install
```

## 開発用サーバー

開発用サーバー起動 `http://localhost:3000`:

```bash
# yarn
yarn dev
```

## 本番

本番ビルド:

```bash
# yarn
yarn build
```
